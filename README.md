# Random-Number-Guessing-Game-Python
This is a simple Python-based number guessing game where the computer randomly selects a number between 1 and 100, and the user tries to guess it.  
The program uses the random module to generate a number between 1 and 100.
The user is asked to enter a number as their guess.
The program compares the user’s guess with the computer’s number:
If the guess is higher, it shows "Too High".
If the guess is lower, it shows "Too Low".
If the guess is correct, it shows "Equal".
